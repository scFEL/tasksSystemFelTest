#### Повний проєкт (клонований репозиторій):
```
npm i
npm run build
npm run dev
```
# Функціонал
- Підключення через MetaMask
- Створення завдання
-- Текст завдання
-- Учасники - перелік адрес, які будуть мати доступ до виконання завдання (якщо порожній, завдання відкрите для всіх)
-- Експерти - перелік адрес, які можуть підтверджувати виконання завдання (автор завдання НЕ включається в список автоматично)
-- Винагорода - сума яка буде автоматично перерахована учаснику, який успішно виконав завдання (для цього також треба поповнити баланс завдання)
-- Дедлайн - дата, до якої треба здати завдання
- Поповнення балансу завдання
-- Поповнювати баланс може будь який користувач (як автор завдання, так і спонсор, волонтер)
- Повернення балансу (якщо він не був вичерпаний до дедлайну завдання)
- Виконання завдання (подача запиту)
-- Має бути вказана текстова відповідь/підтвердження виконання завдання
-- Не можна подавати запит, якщо у вас є нерозглянутий запит на цьому завданні або якщо ви вже виконали це завдання
- Обробка запиту експертом (прийняти/відхилити)
- Перевірка сертифікату
-- Додаткова можливість швидко перевірити виконання завдання
-- Сертифікат має формат (номер завдання):(адреса виконавця)
-- Сертифікат можна скопіювати із вкладки Tasks
